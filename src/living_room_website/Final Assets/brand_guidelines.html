Website Brand Guidelines v1.0

Brand Philosophy

Living Room Baithaks brings classical music into intimate home settings. Our website design reflects this ethos: minimal, bold, accessible, and warm. We use strategic orange accents to create energy without overwhelming, paired with clean typography and generous white space for a sophisticated feel.

Color Palette

Our color system is built around strategic orange usage as an accent, not a dominant color. Orange creates energy and urgency, while gold adds sophistication and warmth.

Responsive Typography

Use clamp() for fluid, mobile-friendly typography that scales perfectly without awkward line breaks. 

/* Artist names - scales from 1.5em to 2em */
.hero-artists {
    font-size: clamp(1.5em, 4vw, 2em);
}

/* Event title - scales from 1.2em to 1.5em */
.hero-title {
    font-size: clamp(1.2em, 3vw, 1.5em);
}

/* Seats count - scales from 1.1em to 1.3em */
.seats-count {
    font-size: clamp(1.1em, 2.5vw, 1.3em);
}

Do's and Don'ts

  • Use orange strategically as accents

  • Use left borders for containers

  • Use clamp() for responsive text
  • Test on mobile devices
  • Use orange dots for icons
  • Keep designs minimal and bold
âœ— DON'T
  • Add orange borders to images
  • Add orange borders to buttons
  • Use emoji icons
  • Create card-style borders
  • Make orange dominant color
  • Use hard-to-read white on white
  • Create line breaks on mobile
  • Overcrowd with decorative elements
